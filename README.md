# AlphaAlgebra: Formula Vault

A modular Python-based CLI tool designed to automate and centralize algebraic calculations.

## Features
- Coordinate Geometry: Calculate Slope, Midpoint, and Distance.
- Input Sanitization: Robust handling of whitespace and character casing.
- Error Management: Comprehensive protection against non-numeric inputs.

## Usage
1. Execute the main interface: python main.py
2. Input the desired formula name (e.g., slope).
3. Provide numerical coordinates when prompted.
